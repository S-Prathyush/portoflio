Flower Type Prediction

This project involves creating a web application that enables users to upload images of flowers and receive predictions on the type of flower depicted. The core of the application is a machine learning model developed using TensorFlow/Keras, based on a Convolutional Neural Network (CNN) architecture. The model has been trained with a dataset of flower images to classify various flower types with high accuracy. The web application, built using Flask, provides a user-friendly interface where users can upload flower images in formats such as JPEG or PNG. Upon uploading, the image is processed and fed into the model, which then predicts the flower type and displays the result in real-time. The system includes features such as image upload, real-time prediction, and a simple, intuitive user interface. This project not only demonstrates the application of machine learning in image classification but also provides a practical tool for flower identification. Future enhancements could involve expanding the flower categories, improving model accuracy with advanced architectures, and enhancing the user interface with additional features.
